# linux命令

## Overview

`command [options] [op-objects]`
+ ex `ls -lah /home ./`

  + "command": `ls`

  + "options": `-lah`

  + "op-objects": `/home ./`

+ `-l`, long listing name

+ `-t`, 按时间排序

+ `-r`, 反向排序
