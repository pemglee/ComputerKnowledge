# 数据库概念

+ 数据库, Database, 关联表(database tables)的集合

  RDBMS (Relational Database Management System), 关系数据库管理系统

  + 数据表, Database Table, 数据的矩阵

    + 列, column, 包含相同类型的数据
    + 行 / 元组 / 记录, row, 一组相关的数据
      + 表头, header, 列的名称
    + 主键, PK (Primary Key),  
    + 外键, FK (Foreign Key),  
    + 复合建, ,
    + 索引, Index,
