# Linux Directory Descriptions

## Overview

+ "/", 根目录

  + "afs"

  + "bin", Binary, 可执行的二进制文件。

  + "boot", 启动Linux的核心文件，包括链接文件和镜像文件

  + "dev", 设备目录（主要是**外接设备**），挂载后可使用。

  + "etc", 系统配置文件等

  + "home", "root"外的用户目录

  + "lib", 系统动态链接库

  + "lib64"

  + "lost+found"

  + "media", 自动识别设备

  + "mnt", 挂载设备

  + "opt", 可选软件

  + "proc", Linux运行时的进程文件

  + "root", root用户的目录

  + "run"

  + "sbin", super binary, 供*root权限用户*执行的可执行文件

  + "selinux"

  + "srv", 系统服务

  + "sys"

  + "tmp", 临时目录

  + "usr", 用户自己安装的应用程序

    + "bin"

    + "sbin"

    + "src"

  + "var", 系统程序日志的存放目录
