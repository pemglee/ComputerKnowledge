# Linux Directory Descriptions

## Overview

+ "/", 根目录

  + "bin", Binary, 可执行的二进制文件。

  + "dev", 设备目录（主要是**外接设备**），挂载后可使用。

  + "etc", 系统配置文件等

  + "home", "root"外的用户目录

  + "proc", Linux运行时的进程文件

  + "mnt", 挂载设备

  + "root", root用户的目录

  + "sbin", super binary, 供*root权限用户*执行的可执行文件

  + "urs", 用户自己安装的应用程序

  + "var", 系统程序日志的存放目录
